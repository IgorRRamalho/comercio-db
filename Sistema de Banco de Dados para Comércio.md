# Sistema de Banco de Dados para Comércio

Este repositório contém o script SQL para a modelagem de um banco de dados relacional simples para um pequeno comércio (livraria/cafeteria).

## Estrutura do Banco de Dados

O banco de dados é composto pelas seguintes tabelas:

1.  **Produtos**: Armazena informações sobre os itens disponíveis.
2.  **Pedidos**: Armazena informações sobre os pedidos dos clientes.
3.  **Itens_Pedido**: Tabela de relacionamento para detalhar os produtos em cada pedido.

O arquivo `schema.sql` contém os comandos `CREATE TABLE` e `INSERT` para popular as tabelas com dados de exemplo.
